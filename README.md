## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/r-pranav-git/Web-Application-Development.git
   ```

2. Open the project folder:

   ```bash
   cd Web-Application-Development
   ```

3. Run HTML files:

   * Open any `.html` file in your browser

4. For PHP files:

   * Use a local server like **XAMPP / WAMP**
   * Place the folder in `htdocs` and run via `localhost`
